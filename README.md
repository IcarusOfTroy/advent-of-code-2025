# Advent of Code

This repository contains my solutions to the Advent of Code programming challenges. Each year, I create a new directory for that year's challenges, and within each directory, I have subdirectories for each day of the event.

I will be writing my solutions in rust, and will include readme files with documentation for each day's challenge.
